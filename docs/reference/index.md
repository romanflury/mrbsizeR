# Package index

## Exported Functions

- [`CImap()`](https://romanflury.github.io/mrbsizeR/reference/CImap.md)
  : Computation of simultaneous credible intervals.
- [`HPWmap()`](https://romanflury.github.io/mrbsizeR/reference/HPWmap.md)
  : Computation of pointwise and highest pointwise probabilities.
- [`MinLambda()`](https://romanflury.github.io/mrbsizeR/reference/MinLambda.md)
  : Numerical optimization for finding appropriate smoothing levels.
- [`TaperingPlot()`](https://romanflury.github.io/mrbsizeR/reference/TaperingPlot.md)
  : Plot of tapering functions.
- [`dctMatrix()`](https://romanflury.github.io/mrbsizeR/reference/dctMatrix.md)
  : Create a n-by-n discrete cosine transform matrix.
- [`dftMatrix()`](https://romanflury.github.io/mrbsizeR/reference/dftMatrix.md)
  : Create a n-by-n discrete Fourier transform matrix.
- [`eigenLaplace()`](https://romanflury.github.io/mrbsizeR/reference/eigenLaplace.md)
  : Generate eigenvalues of discrete Laplace matrix.
- [`eigenQsphere()`](https://romanflury.github.io/mrbsizeR/reference/eigenQsphere.md)
  : Generate eigenvalues of precision matrix Q on the surface of a
  sphere.
- [`fftshift()`](https://romanflury.github.io/mrbsizeR/reference/fftshift.md)
  : Swap the quadrants or halves of a 2d matrix.
- [`ifftshift()`](https://romanflury.github.io/mrbsizeR/reference/ifftshift.md)
  : Inverse FFT shift of a 2d matrix.
- [`mrbsizeRgrid()`](https://romanflury.github.io/mrbsizeR/reference/mrbsizeRgrid.md)
  : Multiresolution analysis of random signals.
- [`mrbsizeRsphere()`](https://romanflury.github.io/mrbsizeR/reference/mrbsizeRsphere.md)
  : Multiresolution analysis of random signals for spherical data.
- [`rmvtDCT()`](https://romanflury.github.io/mrbsizeR/reference/rmvtDCT.md)
  : Sampling from marginal posterior multivariate t-distribution.
- [`tridiag()`](https://romanflury.github.io/mrbsizeR/reference/tridiag.md)
  : Generate a tridiagonal matrix.
- [`turnmat()`](https://romanflury.github.io/mrbsizeR/reference/turnmat.md)
  : Turn matrix 90 degrees counter-clockwise.
